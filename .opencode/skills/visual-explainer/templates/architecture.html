System components and data flow

Layer 1

Client Tier

Web app, mobile, API consumers

  • React SPA — Frontend application
  • REST API — External consumers
Layer 2

Service Tier

Business logic and orchestration

  • api-gateway — Request routing
  • auth-service — Authentication
  • core-service — Main logic
Layer 3

Data Tier

Persistence and caching

  • PostgreSQL — Primary database
  • Redis — Cache layer
Overview

Request Flow

Client requests hit the API gateway, which routes to the appropriate service. Services authenticate via the auth service, execute business logic, and persist to the data tier. Redis caches frequently accessed data.

Key Implementation Details
  • middleware/auth.go — JWT validation middleware
  • services/router.go — Request routing logic
  • db/migrations/ — Schema migrations
